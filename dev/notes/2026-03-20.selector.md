Selector
==============================================================================

How a `Selector` works:

-   Has an `is_match(obj: object) -> bool` method
-   Can be applied to structures to select things
-   Common values and structures have a natural conversion to selectors:
    -   `str` selects that key from mappings and that attribute from other
        objects
    -   `int` selects that index from sequences
    -   `{K: V}` selects objects where the value or attribute selected by `K`
        matches `V`. For instance `{"name": "Data Source"}` would select objects
        that have a `name` attribute or key equal to `"DataSource"`.
    -   Callable `(x: object) -> bool` is a predicate filter


```python
dig(obj, ["etap", "defined_values", {"name": "Data Source"}])

select(obj)[{"name": "blah"}]

selector(name="blah") & selector(lamda _: rand() > 0.5)

selector("any", name="blah", color="blue")
```
